About the Game

Woody 99 and Sudoblock are block puzzle games featuring Sudoku-style boards combined with economy systems. Players drag and drop blocks to complete rows and columns, managing boosters and progression across an extensive level catalogue.

My Responsibilities

  • Developed new game modes extending the core block puzzle mechanics.
  • Maintained and updated existing features across multiple live releases.
  • Built an in-editor level creation tool to streamline the content pipeline.
  • Integrated Firebase for Remote Config, Analytics, and Crashlytics.
  • Managed asset streaming via Addressables and Spine character animations.
